## Project Overview

This project is a responsive landing page for a **Software Development Company**, built from a Figma Community design.

The goal of the project was to convert the original UI/UX design into a clean, responsive, and modern website while maintaining the layout, spacing, typography, and visual hierarchy of the original design.

### Technologies Used

- HTML5
- Tailwind CSS

### Features

- Fully responsive layout
- Modern and clean user interface
- Mobile, tablet, and desktop support
- Reusable and well-structured components
- Optimized spacing and typography
- Fast and lightweight implementation using Tailwind CSS

### Design Source

The UI design was created in **Figma Community** and used as the reference for the frontend implementation.
